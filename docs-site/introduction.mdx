Use this documentation to get started, deploy to production, integrate with the API, and extend Overseer with custom skills and MCP servers.

## Quick Install

One-line install (recommended):

```bash
curl -fsSL https://overseer.sh/install | bash
```

Or clone and run manually:

```bash
git clone https://github.com/Quad-Labs-LLC/overseer.git
cd overseer
pnpm install
pnpm run db:init
cp .env.example .env
# Edit .env with your credentials, then:
pnpm dev
```
